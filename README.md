# Aplikacje Baz Danych - Laboratorium LINQ

Projekt konsolowy zrealizowany w ramach ćwiczeń laboratoryjnych (Ćwiczenia 3). Aplikacja demonstruje praktyczne zastosowanie zapytań LINQ (Language-Integrated Query) w języku C# do przetwarzania kolekcji obiektów w pamięci, co symuluje pracę z bazą danych uczelni.

## Zakres zrealizowanych zadań

W ramach projektu zaimplementowano 20 metod w pliku `ZadaniaLinq.cs`, które pokrywają następujące zagadnienia:
* **Podstawowe operatory LINQ:** `Where`, `Select`, `OrderBy`, `ThenBy`, `FirstOrDefault`.
* **Weryfikacja i agregacja:** `Any`, `All`, `Count`, `Max`, `Average`.
* **Paginacja i unikalność:** `Skip`, `Take`, `Distinct`.
* **Łączenie i grupowanie danych:** `Join`, `GroupJoin`, `GroupBy`.
* **Złożone wyzwania biznesowe:** Kombinacja wielu operatorów do tworzenia zaawansowanych raportów (np. filtrowanie po dacie, zliczanie aktywnych zapisów, wyliczanie średnich ocen).
